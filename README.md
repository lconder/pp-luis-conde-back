# Pasos

1. Clonar el repositorio  
2. Agregar un archivo .env con las siguientes variables (sustituyendo por las tuyas):  
	  
	-PORT=3003  
	-DB_USER=root  
	-DB_PASSWORD=*****  
	-DB_HOST=localhost  
	-DB_PORT=43717  
	-DB_NAME=packnpack  
	
3. npm install
4. node app.js o npm start
5. En tu navegador: localhost:3003
